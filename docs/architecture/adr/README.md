## 治理规则

- 每个 ADR 一个文件，编号递增：`NNN-简短标题.md`
- 状态流转: `提议` → `已采纳` → `已废弃` / `继承`
- ADR 只追加不删除，废弃时标注替代方案
- 主题索引维护在 [topic-map.md](topic-map.md)
- 所有 ADR 须引用相关的代码文件或文档

## 模板
